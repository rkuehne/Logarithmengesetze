# Logarithmengesetze

Interaktives Arbeitsblatt für Klasse 10 zu den Logarithmengesetzen.

## Enthalten

- Regelkarten zu Produkt-, Quotienten- und Potenzgesetz
- Labor mit Zahlenwerten und direktem Vergleich beider Seiten
- Training mit Multiple Choice und Drag-and-Drop-Bausteinen
- Hinweise, Lösungen und Sitzungsstatistik
- Druckansicht für den Unterricht

## GitHub Pages

Das Projekt ist für GitHub Pages aus dem Repository-Root gedacht. Nach dem Push auf `main` wird die `index.html` veröffentlicht.
